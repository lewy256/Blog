Sending an email from a .NET application

(8/17/2024)

This article explains how to send an email from a .NET application using Gmail's SMTP server in three steps: create a Gmail account, enable two-factor authentication, and generate an app password for your .NET application.

● Create a Gmail account and enable two-factor authentication for your Gmail account.

● Generate an app password.

To find the app password generator, you can use the search bar.

Save your code; you will need it later

● Create a console app and paste this code with your credentials.

Remember to hide your app password in a Key Vault or environment variables for security purposes.


        public void SendEmail() {
            var smtpClient = new SmtpClient() {
                Port = 587,
                Credentials = new NetworkCredential("Your email", "Your app password code"),
                EnableSsl = true,
                Host = "smtp.gmail.com"
            };
            smtpClient.Send("Your email", "Receiver email", "Subject", "Content");
        }
